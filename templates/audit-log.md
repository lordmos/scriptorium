# {{书名}} 审计日志

## 日志格式说明

每条记录包含以下字段：

| 字段 | 说明 |
|------|------|
| 日期 | 操作日期，格式 YYYY-MM-DD |
| Phase | 当前所处的项目阶段（Phase 1~7） |
| 操作类型 | 见下方"操作类型分类" |
| 影响范围 | 哪些文件、章节受到影响 |
| 决策 | 做了什么决定 |
| 原因 | 为什么做这个决定 |
| 执行者 | 哪个Agent或人执行的操作 |

### 操作类型分类

<!-- 使用标签分类便于后续筛选 -->

| 标签 | 含义 | 示例 |
|------|------|------|
| `[结构]` | 章节结构变更 | 合并两章、新增章节、调整顺序 |
| `[术语]` | 术语相关决策 | 确定翻译、更换术语 |
| `[比喻]` | 比喻相关决策 | 新增比喻、废弃比喻 |
| `[审查]` | 审查中的重大发现 | 发现事实错误、逻辑问题 |
| `[源码]` | 源码相关变更 | 版本更新、发现新的重要文件 |
| `[风格]` | 写作风格调整 | 修改语气、调整详略 |
| `[读者]` | 读者反馈导致的变更 | 难度调整、增加解释 |
| `[修复]` | 错误修正 | 代码错误、事实错误 |
