# {{書名}} 審計日誌

## 日誌格式說明

每條記錄包含以下字段：

| 字段 | 說明 |
|------|------|
| 日期 | 操作日期，格式 YYYY-MM-DD |
| Phase | 當前所處的項目階段（Phase 1~7） |
| 操作類型 | 見下方"操作類型分類" |
| 影響範圍 | 哪些文件、章節受到影響 |
| 決策 | 做了什麼決定 |
| 原因 | 爲什麼做這個決定 |
| 執行者 | 哪個Agent或人執行的操作 |

### 操作類型分類

<!-- 使用標籤分類便於後續篩選 -->

| 標籤 | 含義 | 示例 |
|------|------|------|
| `[結構]` | 章節結構變更 | 合併兩章、新增章節、調整順序 |
| `[術語]` | 術語相關決策 | 確定翻譯、更換術語 |
| `[比喻]` | 比喻相關決策 | 新增比喻、廢棄比喻 |
| `[審查]` | 審查中的重大發現 | 發現事實錯誤、邏輯問題 |
| `[源碼]` | 源碼相關變更 | 版本更新、發現新的重要文件 |
| `[風格]` | 寫作風格調整 | 修改語氣、調整詳略 |
| `[讀者]` | 讀者反饋導致的變更 | 難度調整、增加解釋 |
| `[修復]` | 錯誤修正 | 代碼錯誤、事實錯誤 |
